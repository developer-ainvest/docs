# 快速上手
## 1.登录官网
访问 [AInvest](https://www.ainvest.com)官网 并点击右上角“登录”按钮，按照提示填写您的基本信息进行登录 （目前平台支持短信登录、邮箱登录，Google, Twitter, Apple 的 OAuth 登录）

## 2.创建API key
进入[创建API密钥](https://ainvest-www.touzime.com/pricing/developer-manage/#api-key)页面，点击“新建API密钥”，创建您的API key

## 3. 通过REST 接口进行服务调用
您可以直接在平台的“[文档链接](https://docs.ainvest.com/api-reference/news/newswire/en/page)”中使用您的 API key 进行在线调用，此处可以生成对应语言的代码。

点击查看[付费计划](https://ainvest-www.touzime.com/pricing/business/)
