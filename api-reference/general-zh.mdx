# 1.请求头

## 1.1 多语言

部分接口例如新闻等，可以支持多种语言，可以通过在请求头中添加 `Accept-Language = zh-hans` 的方式指定语种，一般来说默认语言为英文。

支持语种:

| 语种       | 代码   |
|------------|--------|
| 印尼语     | id     |
| 德语       | de     |
| 日语       | ja     |
| 法语       | fr     |
| 简体中文   | zh-hans|
| 繁体中文   | zh-hant|
| 英语       | en     |
| 西班牙语   | es     |
| 越南语     | vi     |
| 阿拉伯语   | ar     |
| 韩语       | ko     |
| 马来语     | ms     |
| 泰语       | th     |
| 希伯来语   | he     |
| 葡萄牙语   | pt     |
